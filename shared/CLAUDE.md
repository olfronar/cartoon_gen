# Shared Module Internals

- **Data contracts** (`models.py`): `RawItem` -> `ScoredItem` -> `ComedyBrief` (single `items` list) -> `Logline` (includes `format_type`) -> `Synopsis` (includes `world_seed: str = ""` for place history/sensory detail) -> `SceneScript` (11 fields, includes `transformation: str = ""`, `billy_emotion: str = ""`) -> `CartoonScript` (includes `format_type`) -> `ShotResult` -> `ShotsManifest` -> `ClipResult` -> `VideoManifest`. All agents share these. `ComedyBrief.from_dict()` handles both new `items` and legacy `top_picks`/`also_notable` keys for backward compatibility. `Synopsis.from_dict()` accepts both `development` and legacy `escalation` keys for backward compatibility. `SceneScript.from_dict()` handles missing optional fields with backward compat defaults. `Logline.format_type` and `CartoonScript.format_type` default to `""` for backward compatibility.
- **ffmpeg** (`ffmpeg.py`): `run_ffmpeg()`, `probe_video()`, `ENCODE_ARGS` constant (`["-c:v", "libx264", "-preset", "fast", "-c:a", "aac"]`). Extracted from video_designer assembler for reuse by caption_maker.
- **Assembler** (`assembler.py`): `assemble_script_video()`, `assemble_final_video()` and helpers (`_concat_clips`, `_concat_with_glitch`, `_generate_glitch_clip`). Used by video_designer and caption_maker. `video_designer/pipeline/assembler.py` is a thin re-export for backward compatibility.
- **Utilities** (`utils.py`): `strip_code_fences()`, `parse_iso_utc()`, `strip_html()`, `extract_text()`, `extract_json()`, `call_llm_json()`, `call_llm_text()`, `OUTPUT_INDEX_RE`, `find_latest_output_date()`, `find_script_videos()`, `parse_feed_timestamp()`. `extract_json(text, expect=dict|list)` handles LLM responses with surrounding commentary by trying direct parse then bracket extraction. `find_script_videos(target_date, video_dir, prefer_captioned)` consolidates video discovery for caption_maker and tiktok_publisher. `parse_feed_timestamp(entry)` consolidates feedparser timestamp parsing for RSS sources.
- **Context loader** (`context_loader.py`): `load_characters()`, `load_art_style()`, `load_art_materials()`, `build_context_block()`, `build_reference_image_list()`. Used by script_writer, static_shots_maker, and video_designer.
